# Textual Tags (텍스트 관련 태그)
HTML에서 텍스트 콘텐츠를 구조화하고 서식을 지정하는 태그들입니다. 단순히 시각적 표현뿐만 아니라 브라우저와 검색엔진이 콘텐츠의 의미와 구조를 이해할 수 있도록 **시맨틱한 의미**를 부여합니다.

---
## 1. 제목 태그 `h1` ~ `h6`
문서의 제목과 소제목을 정의합니다. 숫자가 작을수록 중요도가 높고 글자 크기가 큽니다.

```html
<h1>가장 큰 제목</h1>
<h2>두 번째 제목</h2>
<h3>세 번째 제목</h3>
<h4>네 번째 제목</h4>
<h5>다섯 번째 제목</h5>
<h6>가장 작은 제목</h6>
```
**주의**: `h1`은 페이지당 하나만 사용하는 것이 SEO 관점에서 권장됩니다.

---
## 2. `<title>`
브라우저 탭에 표시되는 페이지 제목을 정의합니다. `<head>` 안에 위치하며 화면에는 표시되지 않습니다.

```html
<head>
	<title>tunachoi-dev</title>
</head>
```

---
## 3. `<p>`
단락(Paragraph)을 정의합니다. 텍스트 본문 작성 시 가장 기본적으로 사용하는 태그입니다.

```html
<p>이것은 하나의 문단입니다.</p>
<p>이것은 다른 문단입니다.</p>
```

---
## 4. `<hr>`
수평선(Horizontal Rule)을 삽입합니다. 콘텐츠 간의 구분선 역할을 하며 닫는 태그가 없는 void element입니다.

```html
<p>첫 번째 섹션</p>
<hr />
<p>두 번째 섹션</p>
```

---
## 5. `<br>`
줄바꿈(Line Break)을 삽입합니다. `<p>` 태그와 달리 새로운 단락을 만들지 않고 단순히 줄만 바꿉니다. void element입니다.

```html
<p>1번 줄<br />2번 줄<br />3번 줄</p>
```

---
## 6. `<b>` / `<strong>`
텍스트를 **굵게** 표시합니다. 둘 다 시각적으로 동일하지만 의미가 다릅니다.

`<b>`: 단순히 굵게 표시 (시각적 목적)
`<strong>`: 중요한 내용임을 강조 (시맨틱 의미 있음)

```html
<b>단순 굵게</b>
<strong>중요한 내용 강조</strong>
```
	시맨틱 관점에서 <strong>사용을 권장.

---
## 7. `<pre>`
미리 서식이 지정된 텍스트(Preformatted)를 표시합니다. 공백과 줄바꿈을 **그대로** 유지해서 출력합니다.

```html
<pre>
이 텍스트는
	들여쓰기와 줄바꿈이
		그대로 유지됩니다.
</pre>
```

---
## 8. `<i>`/`<em>`
텍스트를 기울임꼴로 표시합니다. `<b>`/`<strong>`과 동일한 구조입니다.
`<i>`: 단순히 기울임 표시 (시각적 목적)
`<em>`: 강조(Emphasis)의 의미 (시맨틱 의미 있음)

```html
<i>단순 기울임</i>
<em>강조하는 기울임</em>
```

---
## 9. `<mark>`
텍스트를 형광펜으로 표시한 것처럼 **하이라이트** 처리합니다.

```html
<p>시험에서 <mark>이 부분</mark>이 중요합니다.</p>
```

---
## 10. `<sub>`/`<sup>`
`<sub>`: 아래 첨자 (Subscript)
`<sup>`: 위 첨자 (Superscript)

```html
<p>물의 화학식: H<sub>2</sub>O</p>
<p>제곱: x<sup>2</sup></p>
```

---
## 11. `<a>` (Links)
다른 페이지나 URL로 이동하는 하이퍼링크를 만듭니다. `href` 속성에 이동할 주소를 지정합니다.

```html
<!-- 외부 링크 -->
<a href="https://www.github.com/tunachoi-dev" target="_blacnk">tunachoi-dev</a>

<!-- 내부 링크 -->
<a href="/about.html">소개 페이지</a>
```
	target="_black"는 링크를 새 탭에서 여는 속성입니다.

---
# 정리

| 태그               | 용도           |
| ---------------- | ------------ |
| `h1` ~ `h6`      | 제목 (중요도 순)   |
| `<title>`        | 브라우저 탭 제목    |
| `<p>`            | 문단           |
| `<hr>`           | 수평 구분선       |
| `<br>`           | 줄바꿈          |
| `<b>`/`<strong>` | 굵게 (시각/시맨틱)  |
| `<pre>`          | 서식 유지 텍스트    |
| `<i>`/`<em>`     | 기울임 (시각/시맨틱) |
| `<mark>`         | 하이라이트        |
| `<sub>`/`<sup>`  | 아래/위 첨자      |
| `<a>`            | 하이퍼링크        |
