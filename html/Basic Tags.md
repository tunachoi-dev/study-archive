# HTML Basic Tags
## HTML 문서의 기본 구조
HTML 문서는 반드시 정해진 기본 태그들로 골격을 이루어야 합니다.
이 구조는 브라우저가 문서를 올바르게 해석하고 렌더링하기 위한 필수 규칙입니다.

```html
<!ODCTYPE html>
<html lang="ko">
	<head>
		<meta charset="UTF-8" />
		<title>페이지 제목</title>
	</head>
	<body>
		<!-- 화면에 표시될 내용 -->
	</body>
</html>
```
---
## 2. 핵심 태그 역할

| 태그                | 역할                                                      |
| ----------------- | ------------------------------------------------------- |
| `<!DOCTYPE html>` | HTML5 문서임을 브라우저에 선언하는 선언문. 태그가 아니며 문서 최상단에 반드시 위치.      |
| `<html>`          | 문서의 루트(root) 요소. 모든 HTML 요소는 이 태그 안에 포함됨.               |
| `<head>`          | 브라우저 화면에 보이지 않는 문서의 메타데이터를 담는 영역                        |
| `<body>`          | 실제로 브라우저 화면에 표시되는 모든 콘텐츠가 들어가는 영역                       |
| `<meta>`          | 문서에 대한 부가 정보 제공. `charset="UTF-8"`으로 한글 등 다국어 문자 깨짐 방지. |
| `<title>`         | 브라우저 탭에 표시되는 페이지 제목 정의. `<head>` 안에 위치.                 |

---
## 3. 태그별 상세 설명
### \<!DOCTYPE HTML>
태그가 아닌 **선언문(Declaration)** 입니다.
브라우저에게 이 문서가 HTML5 규격임을 알려주는 역할을 합니다.

**주의:** 반드시 문서의 **첫 번째 줄**에 위치해야 합니다. 없을 경우 브라우저가 호환 모드(Quirks Mode)로 동작해 레이아웃이 틀어질 수 있습니다.
### html
문서 전체를 감싸는 최상위(루트) 요소입니다.
모든 HTML 태그는 `<html>` 안에 위치해야 합니다.

`lang` 속성을 지정하면 검색엔진(SEO)과 스크린 리더(접근성)에 유리합니다.

```html
<html lang="ko">
```
### head
브라우저 화면에 직접 표시되지 않는 문서의 정보(메타데이터)를 담는 영역입니다.
`<head>` 안에 들어오는 대표적인 요소들:
- `<meta>` - 문자 인코딩, 뷰포트 설정 등
- `<title>` - 브라우저 탭 제목
- `<link>` - CSS 파일 연결
- `<script>` - JavaScript 파일 연결

```html
<head>
  <meta charset="UTF-8" />
  <title>tunachoi-dev</title>
  <link rel="stylesheet" href="style.css" />
  <script src="app.js"></script>
</head>
```
### body
사용자가 브라우저에서 실제로 보게 되는 모든 콘텐츠가 위치하는 영역입니다.
텍스트, 이미지, 버튼, 링크 등 화면에 표시되는 모든 요소는 `<body>` 안에 작성합니다.
```html
<body>
  <h1>제목</h1>
  <p>본문</p>
  <button>버튼</button>
</body>
```
### \<meta charset="UTF-8">
문서의 문자 인코딩 방식을 지정합니다.
**UTF-8**은 한글, 영어, 일본어 등 전 세계 거의 모든 문자를 표현할 수 있는 표준 인코딩 방식입니다.

**주의**: 이 선언이 없으면 한글 등 비영어 문자가 깨져 보일 수 있습니다.

---
## 4. 정리
HTML 문서는 항상 이 순서와 구조를 지켜야 합니다.
```html
1. <!DOCTYPE html>   →  HTML5 선언 (문서 최상단)
2. <html>            →  루트 요소 시작
3.   <head>          →  메타데이터 영역
4.     <meta>        →  문자 인코딩 등 설정
5.     <title>       →  탭 제목
6.   </head>
7.   <body>          →  화면에 표시될 콘텐츠
8.   </body>
9. </html>           →  루트 요소 종료
```
