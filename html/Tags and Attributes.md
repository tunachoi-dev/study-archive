# 태그(Tag)란 무엇인가?
**태그(Tag)**: 웹 페이지의 구성 요소가 무엇인지 브라우저에게 알려주는 **지시어** 또는 **표식**
## 태그(Tag)의 기본 구조
- **시작 태그 (Opening Tag):** 태그 이름을 괄호로 감싸는 형태. (예: `<p>`)
- **종료 태그 (Closing Tag)**: 태그 이름을 괄호로 감싸는 형태에서 태그 이름 앞에 **슬래시(`/`)** 를 붙임 (예: `</p>`)
- **요소 (Element)**: 시작 태그부터 종료 태그, 그리고 그 사이의 내용 전체를 의미
### 예시
```HTML
<p>안녕하세요, tunachoi-dev입니다.</p>
```
## 주요 특징
- **대소문자 구분 없음**: `<P>`와 `<p>`는 동일하게 작동, 웹 표준(W3C)은 **소문자** 사용을 권장
- **빈 태그 (Empty Tags)**: 일부 태그는 감쌀 내용이 없어서 종료 태그가 필요 없음.
	- 예: `<br>` (줄바꿈), `<img>` (이미지), `<hr>` (수평선)
- **속성 (Attributes)**: 태그에 추가적인 정보를 제공. 항상 시작 태그 안에 작성
	- 예: `<img src="image.jpg">` (이미지의 출처를 알려줌)
- **중첩 규칙 (Nesting Rules)**: 태그 안에 다른 태그를 넣을 수 있음. 태그 안에 다른 태그를 넣을 때는 **나중에 열린 태그가 먼저 닫혀야 함**
	- 예: `<div><p>내용</p></div>`
## 주요 태그(Tag)
태그는 엄청나게 다양하기 때문에 다 외우지 말고 필요할 때 검색을 하자.
- `<div>`: 구역을 나눌 때 사용
- `<section>`: 문서의 콘텐츠 사분면이나 주제별 그룹을 정의할 때 사용
- `<header>` / `<footer>`: 페이지나 섹션의 머리말과 바닥글을 정의
- `<nav>`: 메뉴, 목차 등 내비게이션 링크를 모아둘 때 사용.
- `<main>`: 해당 문서의 핵심적인 본문 내용을 담는 태그
- `<h1>` ~ `<h6>`: 제목을 나타낼 때 사용. 숫자가 클 수록 폰트가 작음. `<h1>`은 페이지당 하나만 사용하는 것이 관례
- `<a>`: 링크를 걸 때 사용, `href` 속성으로 목적지를 지정
- `<ul>` & `<li>`: 순서가 없는 목록(불렛 포인트)을 만들 때 세트로 사용. 메뉴 바를 만들 때도 필수
- `<ol>` & `<li>`: 순서가 있는 목록(숫자)을 만들 때 사용
- `<form>`: 입력 데이터들을 서버로 보낼 때 감싸는 틀
- `<input>`: 텍스트 입력, 체크박스, 라디오 버튼 등 다양한 형태의 입력을 받음
- `<button>`: 클릭 가능한 버튼을 생성

# 속성(Attributes)
**속성(Attributes)**: 태그에 **추가적인 정보**를 제공하는 설정값. 태그가 "무엇"인지 정의한다면, 속성은 그 태그가 "어떻게" 작동하거나 어디와 연결되는지를 결정.
## 속성(Attributes)의 기본 규칙
속성은 반드시 다음과 같은 규칙을 따름.
- **시작 태그** 안에서만 작성 (종료 태그에는 쓰지 않음)
- **이름="값"** (Name="Value")의 쌍으로 이루어짐
- 여러 개의 속성을 사용할 때는 **공백(띄어쓰기)** 으로 구분
	- **예시**:
		- `<a href="https://www.github.com/tunachoi-dev" target="_blank">tunachoi-dev로 이동</a>`
		- `href`: 연결할 주소를 알려주는 속성 이름
		- `"https://www.github.com/tunachoi-dev"`: 실제 주소 값
		- `target`: 링크를 어디서 열지 결정하는 속성 이름
		- `"_blank"`: 새 탭에서 열라는 값
## 주요 속성(Attributes)
속성도 종류가 많기 때문에 필요할 때마다 검색을 통해 찾아보자.
- `id`: 요소에 부여하는 **고유한 이름** (페이지 내 단 하나)
	- 예: `<div id="header">`
- `class`: 여러 요소에 공통으로 부여하는 **그룹 이름**
	- 예: `<p class="red-text">`
- `src`: 이미지나 스크립트 파일의 **경로(Source)**
	- 예: `<img src="logo.png">`
- `href`: 클릭 시 이동할 **링크 주소**
	- 예: `<a href="index.html">`
- `style`: HTML 태그에 직접 **CSS 디자인** 적용
	- 예: `<span style="color:blue;">`
